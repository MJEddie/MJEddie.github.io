Markdown語法

Markdown 語法是什麼？

  • Markdown 是一種輕量級標記式語言，為了實現易讀易寫的特性 Markdown 語法全由標點符號所組成，讓它們看起來就像是所要表達的意思。
  • 因為其易讀易寫的特性，目前許多網站都廣泛使用 Markdown 語法。

三、清單

Markdown 支援無序清單和有序清單，要在清單下加入段落只要 +tab 就好。
無序清單可使用星號、加號或是減號作為標記：

Markdown語法：

顯示結果：

  • 無序清單
  • 這也是清單
  • 這還是清單

有序清單則使用數字接一個英文句點：

Markdown語法：

顯示結果：

  1. 有序清單１
  2. 有序清單２
